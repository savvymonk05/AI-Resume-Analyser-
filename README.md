## ✨ Features

| Feature | Description |
|--------|------------|
| 📊 Match Score | ATS compatibility percentage |
| ❌ Missing Skills | Identify skill gaps instantly |
| ✅ Strengths | Highlight your strong points |
| 💬 Feedback | Recruiter-style evaluation |
| 🚀 Suggestions | Actionable improvement steps |
| 📈 History | Stored in Google Sheets |

## 🤔 Why I Built This

Most resumes fail ATS screening before reaching recruiters.

I wanted to build a system that:
- Gives real-time feedback
- Helps candidates improve instantly
- Simulates recruiter evaluation using AI

 ## ⚡ Quick Start

1. Import n8n workflow made by me 
2. Add Gemini API key  
3. Run n8n  
4. Open `index.html`  
5. Upload resume & analyze  

Done ✅
